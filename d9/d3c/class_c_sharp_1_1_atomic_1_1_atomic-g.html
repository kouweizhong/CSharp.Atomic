Detailed Description

template<T>
class CSharp::Atomic::Atomic< T >

Provides lock-free atomic read/write utility for a reference type, T, instance.

The atomic classes found in this package were are meant to replicate the java.util.concurrent.atomic package in Java by Doug Lea. The two main differences are implicit casting back to the T data type, and the use of a non-volatile inner variable.

The internals of these classes contain wrapped usage of the System.Threading.Interlocked class, which is how we are able to provide atomic operation without the use of locks.
