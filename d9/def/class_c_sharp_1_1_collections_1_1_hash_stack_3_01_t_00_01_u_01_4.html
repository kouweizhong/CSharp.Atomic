Detailed Description

This collection implementation uses an internal Dictionary as a hash table using Stack instances for buckets.

The Push operation lazily creates a Stack for the instance if one does not already exist for the key

Note that this class is not thread safe by default. This is not a requirement for it's use in LockCondition, since each waiting thread releases one at a time.
