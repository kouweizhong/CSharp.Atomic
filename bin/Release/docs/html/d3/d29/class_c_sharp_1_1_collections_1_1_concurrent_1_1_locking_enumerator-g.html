Detailed Description

template<T>
class CSharp::Collections::Concurrent::LockingEnumerator< T >

This class serves as a thread-safe wrapper for an IEnumerator implementation.

It locks as soon as the instance is created and will release the lock once disposed.

Template Parameters:
TThe type of the enumerator.
