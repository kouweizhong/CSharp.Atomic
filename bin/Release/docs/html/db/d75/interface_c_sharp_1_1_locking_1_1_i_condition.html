Detailed Description

This interface defines a condition instance that can be used for flow control within a lock.

For instance, two conditions can be used to created a bounded buffer where one condition waits while the buffer is empty, and the other waits when the buffer is full. When items are removed from the buffer, the condition waiting on full will be signaled. When items are added to the buffer, the condition waiting on empty will be signaled.
