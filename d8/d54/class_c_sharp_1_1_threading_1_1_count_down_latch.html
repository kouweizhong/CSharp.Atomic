Detailed Description

This class acts as a lock-less, thread safe count down which can be used to perform multiple async tasks and await signals from all threads without having to wait for the thread to terminate.

It's based on the java.util.concurrent.CountDownLatch

class found in Java.

This class performs a very similar task to the CountDownEvent class from .NET 4.5, but is compatible with .NET 3.0+ for use with platforms like Unity3D.
