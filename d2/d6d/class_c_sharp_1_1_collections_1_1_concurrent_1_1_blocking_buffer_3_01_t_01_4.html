Detailed Description

This collection is similar to the BlockingQueue.

However, in addition to blocking the accessing thread on a Dequeue() when the buffer is empty, it will also block the accessing thread on an Enqueue() when the buffer has reached it's max capacity.

Currently, this class and BlockingQueue are nearly identical, with the only difference being how the ICondition instances within BlockingBuffer<T> controling both the upper and lower bounds of the queue, where the BlockingQueue<T> only controls the lower bound.

While it would be possible to use inheritance here, it may be beneficial to keep the classes separate for now.
